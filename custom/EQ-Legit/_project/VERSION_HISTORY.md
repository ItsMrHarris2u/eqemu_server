# EQ‑Legit — Version History

This file provides a chronological overview of major EQ‑Legit versions and the
scope of each release.

---

## Version 0.3.0 — Taxonomy Standardization
**Focus:** Renaming modules and unifying documentation  
- Introduced Loot_Items, Quest_Items, Orphan_Items  
- Standardized /docs folders across all modules  
- Updated project-level documentation  
- Prepared repository for future expansions (Kunark, Velious, Custom)

---

## Version 0.2.0 — Legacy Loot & Orphan NPC Restoration
**Focus:** Rebuilding missing ecosystems and restoring loot  
- Rebuilt CT mosquito NPCs and spawns  
- Restored Rubicite loot system  
- Normalized SQL structure for LegacyItems and CT Mosquitos  
- Added complete documentation sets

---

## Version 0.1.0 — Project Foundation
**Focus:** Initial structure and standards  
- Created EQ‑Legit repository  
- Added initial LegacyItems module  
- Established universal SQL numbering (01–10)  
- Added project-level documentation placeholders

---

Future versions will expand EQ‑Legit into Kunark, Velious, and custom
restoration modules while maintaining the same modular architecture.
