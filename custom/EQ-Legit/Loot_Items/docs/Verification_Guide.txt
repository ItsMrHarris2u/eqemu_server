EQ‑Legit — Loot Items Module
Verification Guide

This guide explains how to verify that the Loot Items Module is correctly
installed and functioning.

Step 1 — Verify Items
SELECT id, name FROM items
WHERE id IN (list of restored legacy item IDs);

Step 2 — Verify Lootdrops
SELECT id, name FROM lootdrop
WHERE id BETWEEN <module range start> AND <module range end>;

Step 3 — Verify Lootdrop Entries
SELECT lootdrop_id, item_id, chance
FROM lootdrop_entries
WHERE lootdrop_id BETWEEN <range>;

Step 4 — Verify Loottables
SELECT id, name FROM loottable
WHERE id BETWEEN <range>;

Step 5 — Verify NPC Wiring
SELECT npc_id, loottable_id
FROM npc_loot
WHERE loottable_id BETWEEN <range>;

Step 6 — In-Game Testing
- Kill target NPCs
- Confirm legacy items drop at expected rarity
- Confirm no unintended NPCs drop legacy items

If all checks pass, the Loot Items Module is fully operational.
